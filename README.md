# coralbloom

An infinite reaction-diffusion fluid animation rendered in the terminal using
24-bit color and Unicode half-blocks. Patterns drift and never repeat.

## Install (Homebrew)

```sh
brew tap jverene/coralbloom
brew install coralbloom
coralbloom            # Ctrl-C to quit
```

## Usage

```sh
coralbloom            # run the animation (Ctrl-C to quit)
coralbloom --version
coralbloom --help
```

Requires a terminal with 24-bit (truecolor) support.

## Run from source

```sh
python3 core.py
```
